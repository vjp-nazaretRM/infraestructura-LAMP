# Changelog

Todas las novedades notables de este proyecto seran documentadas en este archivo. El formato esta basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/) y este proyecto se adhiere a [Versionado Semantico](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-25
### Anadido
- **HAProxy**: Anadida configuracion tecnica del balanceador de carga en capa 7 como proxy inverso corporativo.
- **REVISION.md**: Reflexion completa y honesta del equipo sobre la resolucion de conflictos Git y el intercambio de roles.
- **Mermaid**: Integracion de diagramas Gantt y de red actualizados en alta definicion dentro de la documentacion.

## [0.3.0] - 2026-05-25
### Anadido
- **Servidor Web y BD**: Manuales completos de instalacion de Apache, PHP y MariaDB con separacion estricta de usuarios y bases de datos.
- **SSH & Firewall**: Implementada securizacion avanzada de SSH en puerto 2222 y combinacion de reglas UFW.
- **Conflictos Git**: Resolucion forzada de conflictos mediante Git Rebase de la rama `feature/guia-operacion` para asegurar un historial lineal.

## [0.2.0] - 2026-05-25
### Anadido
- **Analisis y Diseno**: Incorporacion de requisitos de red de hardware y tabla inicial de versiones de software.
- **Conflictos Git**: Primer conflicto forzado resuelto mediante fusion tradicional (Git Merge) en el archivo `02-diseno.md` incorporando Apache 2.4.60 y soporte SSL con Certbot.

## [0.1.0] - 2026-05-25
### Anadido
- Estructura inicial del repositorio y tareas.
- Borradores iniciales de monitorizacion con Netdata y politicas de backups locales y externas.
